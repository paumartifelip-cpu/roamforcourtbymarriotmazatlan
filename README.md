# Roam × Courtyard by Marriott Mazatlán Beach Resort

Este repositorio contiene la propuesta comercial e interactiva de **Roam** personalizada exclusivamente para la dirección general y el equipo de marketing del **Courtyard by Marriott Mazatlán Beach Resort** (ubicado en Mazatlán, Sinaloa).

La propuesta plantea una colaboración de **Intercambio Inteligente (Zero-Cash)**, donde el hotel facilita noches de hospedaje (en periodos específicos de baja ocupación) a cambio del diseño, implementación técnica y capacitación práctica de sistemas de **Inteligencia Artificial** y **automatizaciones operativas**.

---

## 🎨 Características de Diseño y Estilo

El sitio web está construido bajo una línea de **diseño altamente corporativo, minimalista y geométrico (sin bordes redondeados)**:
- **Ángulos Rectos (100% Recto)**: Todo elemento en el layout (tarjetas, botones, distintivos, listados y tablas) tiene esquinas rectas perfectas para reflejar formalidad, estructura y seriedad empresarial.
- **Tipografía de Prestigio**:
  - **Montserrat**: Para encabezados, títulos y logotipos, aportando una presencia geométrica y corporativa sólida.
  - **Inter**: Para todo el cuerpo de texto, garantizando una excelente legibilidad.
- **Paleta de Colores Inspirada en la Identidad Marriott**:
  - **Azul Marino Marriott** (`#002D62`): Aporta profesionalismo, confianza y estatus corporativo.
  - **Dorado Atardecer** (`#C5A880` / `#B3946B`): Evoca la luz solar de la costa de Mazatlán.
  - **Fondo Limpio**: Espacios en blanco generosos y fondos off-white (`#F8F9FC`) sin emojis ni elementos distractores flotantes.

---

## 🛠️ Pilares Operativos de la Propuesta

1. **Formación en IA para el Equipo**: Capacitación práctica al staff en el uso de IA Generativa para agilizar la redacción de correos bilingües y respuestas a reseñas en TripAdvisor y Marriott Bonvoy.
2. **Automatizaciones de Atención**: Flujos semiautomatizados e inteligentes para WhatsApp y mensajes directos que resuelven dudas recurrentes de Day Passes, horarios del restaurante *Don Joaquin*, salones y amenidades.
3. **Estrategia de Venta Directa**: Optimización de la comunicación para motivar a los huéspedes a reservar en la web oficial del hotel, reduciendo el pago de comisiones de OTAs (Booking, Expedia).
4. **Dirección de Contenido Local**: Guiones y estrategias de contenido vertical enfocadas en resaltar los atardeceres y la privilegiada ubicación en la Zona Dorada.

## 🚀 ¿Cómo visualizar la propuesta localmente?

Dado que la propuesta está contenida en un único archivo HTML optimizado, ligero y autocontenido (`index.html`), se puede previsualizar inmediatamente sin necesidad de instalar dependencias adicionales:

1. Clona este repositorio:
   ```bash
   git clone https://github.com/paumartifelip-cpu/roamforcourtbymarriotmazatlan.git
   ```
2. Accede al directorio:
   ```bash
   cd roamforcourtbymarriotmazatlan
   ```
3. Abre el archivo en tu navegador web de preferencia (se recomienda Google Chrome):
   ```bash
   open index.html
   ```
